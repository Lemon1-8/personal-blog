## 1. Web 前台路由（Next.js App Router）

### 1.1 路由总表

| 路径 | 页面 | 类型 | 鉴权 | 说明 |
|------|------|------|------|------|
| `/` | 首页 | 静态 | 否 | 置顶文章、最新文章、侧边栏热门标签 |
| `/articles` | 文章列表 | 动态 | 否 | 全部已发布文章，支持筛选/排序/翻页 |
| `/articles/[slug]` | 文章详情 | 动态 | 否 | 文章正文、作者、标签、点赞收藏 |
| `/categories` | 分类列表 | 动态 | 否 | 所有分类及其文章数 |
| `/categories/[slug]` | 分类文章 | 动态 | 否 | 该分类下所有文章 |
| `/tags` | 标签列表 | 动态 | 否 | 所有标签及其文章数 |
| `/tags/[slug]` | 标签文章 | 动态 | 否 | 该标签下所有文章 |
| `/search` | 搜索结果 | 动态 | 否 | 关键词搜索，Query: `?q=xxx` |
| `/about` | 关于页 | 静态 | 否 | 博主介绍 |
| `/auth/login` | 登录 | 静态 | 否 | 登录表单 |
| `/auth/register` | 注册 | 静态 | 否 | 注册表单 |
| `/*` | 404 页面 | 静态 | 否 | 未找到 |

### 1.2 页面功能描述

#### 首页 `/`
```
布局:
┌──────────────────────────────────┐
│            Header                │
│  Logo | 导航 | 搜索 | 登录按钮    │
├──────────────────────────────────┤
│  ┌────────────────┐  ┌────────┐ │
│  │  置顶文章 x3     │  │  热门  │ │
│  │  (卡片式)       │  │  标签   │ │
│  │                │  │        │ │
│  │  最新文章 x10   │  │  分类   │ │
│  │  (列表/卡片)    │  │  统计   │ │
│  │                │  │        │ │
│  │  加载更多 /     │  └────────┘ │
│  │  翻页          │            │
│  └────────────────┘            │
├──────────────────────────────────┤
│            Footer                │
└──────────────────────────────────┘
```

- 首次加载显示 10 篇文章，点击"加载更多"追加 10 篇（或使用分页）
- 置顶文章显示前 3 篇（`is_pinned=true`）
- 侧边栏：热门标签（文章数 top 10）、分类统计

#### 文章列表 `/articles`
- 支持查询参数: `?page=1&category=tech&tag=python&sort=views_desc`
- 顶部显示当前筛选条件（面包屑/标签），可清除
- 展示模式：列表模式（大图+摘要）或网格模式（小卡片）
- 底部 Bootstrap 风格分页器

#### 文章详情 `/articles/[slug]`
```
┌──────────────────────────────────┐
│  面包屑: 首页 > 技术 > 文章标题   │
├──────────────────────────────────┤
│  封面大图 (如有)                  │
│  标题                            │
│  作者头像 + 昵称 + 发布时间       │
│  分类标签 + 文章标签            │
│  浏览量 + 点赞数 + 收藏数        │
│  ─────────────────────────────  │
│  文章正文 (富文本 HTML 渲染)     │
│  ─────────────────────────────  │
│  点赞按钮 ❤️ | 收藏按钮 🔖      │
│  ─────────────────────────────  │
│  上一篇 / 下一篇 导航            │
│  ─────────────────────────────  │
│  相关文章 (同分类/同标签)        │
└──────────────────────────────────┘
```

- 浏览量 +1 逻辑：同一 IP 或同一用户 5 分钟内不重复计数
- 已登录用户显示是否已点赞/收藏
- 富文本内容直接渲染 HTML（需做 XSS 过滤）

#### 分类列表 `/categories`
- 分类以卡片网格排列
- 每个分类卡片显示：名称、描述、文章数
- 可点击进入对应分类文章列表

#### 分类文章 `/categories/[slug]`
- 功能同 `/articles?category=xxx`
- 顶部显示分类名称和描述

#### 标签列表 `/tags`
- 标签云/列表布局
- 标签大小随文章数变化（越大文章越多）
- 点击进入对应标签文章列表

#### 标签文章 `/tags/[slug]`
- 功能同 `/articles?tag=xxx`

#### 搜索 `/search?q=xxx`
- 搜索输入框（已预填 query）
- 搜索结果列表（标题+摘要匹配）
- 空结果展示"未找到相关内容"
- 防抖搜索：输入停止 300ms 后发起请求

#### 关于 `/about`
- 纯展示页，内容由后台网站配置中提取

#### 登录 `/auth/login`
```
┌────────────────────┐
│       登录         │
│                    │
│  用户名/邮箱 ├────┤│
│                    │
│  密码      ├────┤│
│                    │
│  [登录]           │
│                    │
│  没有账号？去注册  │
└────────────────────┘
```

- 登录成功 → 跳转回原页面或首页
- 登录成功 → 存储 Token 到 localStorage
- 错误提示：用户名或密码错误

#### 注册 `/auth/register`
```
┌────────────────────┐
│       注册         │
│                    │
│  用户名   ├────┤│
│                    │
│  邮箱     ├────┤│
│                    │
│  密码     ├────┤│
│                    │
│  确认密码 ├────┤│
│                    │
│  [注册]           │
│                    │
│  已有账号？去登录  │
└────────────────────┘
```

- 前端校验密码强度
- 注册成功 → 自动登录跳转首页

#### 404 页面
- 展示"页面未找到"及返回首页按钮

## 2. Web 后台管理路由

所有后台路由前缀 `/admin`，需要登录且角色为 `admin`。

### 2.1 路由总表

| 路径 | 页面 | 说明 |
|------|------|------|
| `/admin` | 控制台 Dashboard | 数据概览：文章数/用户数/浏览量 |
| `/admin/articles` | 文章管理 | 表格展示所有文章（含草稿） |
| `/admin/articles/new` | 新建文章 | 富文本编辑器 |
| `/admin/articles/[id]` | 编辑文章 | 同新建文章，预填数据 |
| `/admin/categories` | 分类管理 | CRUD 操作 |
| `/admin/tags` | 标签管理 | 批量创建/删除 |
| `/admin/media` | 媒体库 | 图片网格，上传/删除 |
| `/admin/users` | 用户管理 | 用户列表，编辑/删除 |
| `/admin/settings` | 网站设置 | 名称、描述、社交链接 |

### 2.2 页面功能描述

#### 控制台 `/admin`
```
┌──────┬──────┬──────┬──────┐
│ 文章  │ 草稿  │ 用户  │ 本月  │
│ 总数  │ 数量  │ 总数  │ 浏览  │
│ 42   │ 3    │ 2    │ 1.2k │
├──────┴──────┴──────┴──────┤
│  最近文章 (表格，5条)      │
│  标题         | 状态 | 操作  │
│  ...         | ...  | 编辑 │
├────────────────────────────┤
│  最近评论 (预留)            │
└────────────────────────────┘
```

#### 文章管理 `/admin/articles`
- 表格显示：标题、分类、状态、浏览量、发布时间、操作（编辑/删除）
- 顶部搜索框（按标题搜索）
- 状态筛选：全部 / 已发布 / 草稿
- 批量操作：批量发布/删除

#### 新建/编辑文章 `/admin/articles/new` `/admin/articles/[id]`
```
┌────────────────────────────────────┐
│  标题 ─────────────────────────── │
│  Slug ───────────────── (自动生成) │
│  摘要 ─────────────────────────── │
│                                    │
│  正文 (TipTap 富文本编辑器)        │
│  ┌──────────────────────────────┐ │
│  │  B I U H1 H2 | 图片 | 代码  │ │
│  │                              │ │
│  │  编辑区域                     │ │
│  │                              │ │
│  └──────────────────────────────┘ │
│                                    │
│  封面图 [选择图片]  [上传]         │
│  分类  [下拉选择]                  │
│  标签  [输入/选择]                 │
│  置顶  [☐]                        │
│                                    │
│  [保存草稿]  [发布]               │
└────────────────────────────────────┘
```

编辑流程：
1. 点击新增 → 进入空编辑器
2. 点击编辑 → 预填已有数据
3. 保存草稿 → 状态保持 draft，不对外可见
4. 发布 → 状态改为 published，设置 published_at
5. TipTap 编辑器集成图片上传（弹窗选媒体库或直接拖拽上传）

#### 分类管理 `/admin/categories`
- 表格：名称、Slug、文章数、排序、操作
- 行内编辑或弹窗编辑
- 新建/编辑弹窗：名称、Slug(自动)、描述、排序
- 删除：如果有文章则禁止删除

#### 标签管理 `/admin/tags`
- 表格：名称、Slug、文章数、操作
- 新建：输入名称、自动生成 Slug
- 删除：级联删除关联

#### 媒体库 `/admin/media`
- 瀑布流/网格布局展示图片
- 每个图片：缩略图、文件名、大小、上传时间
- 点击查看大图和详细信息
- 上传按钮（支持多选）
- 删除按钮（确认弹窗）
- 复制 URL 按钮

#### 用户管理 `/admin/users`
- 管理员可见
- 表格：用户名、邮箱、昵称、角色、注册时间、操作
- 编辑弹窗：角色修改、禁用/启用账号
- 删除用户

#### 网站设置 `/admin/settings`
- 表单：网站名称、描述、Logo URL、Favicon URL
- 社交链接：GitHub、Twitter、微博
- 自定义页脚 HTML
- 保存后立即生效（API 刷新配置缓存）

## 4. 小程序（Taro）路由

### 4.1 路由总表

| 路径 | 页面 | 说明 |
|------|------|------|
| `/pages/index/index` | 首页 | 文章流 |
| `/pages/articles/list` | 文章列表 | 分类/标签筛选 |
| `/pages/articles/detail` | 文章详情 | 带分享功能 |
| `/pages/categories/list` | 分类列表 | |
| `/pages/tags/list` | 标签列表 | |
| `/pages/search/search` | 搜索页 | |
| `/pages/my/index` | 个人中心 | 收藏/登录 |

### 4.2 页面功能描述

#### 首页
- 顶部搜索入口（点击跳转搜索页）
- Tab 切换：推荐 / 最新 / 分类
- 文章卡片流（瀑布流或列表）
- 下拉刷新，上拉加载

#### 文章详情
- 富文本渲染（Taro 内置 rich-text 组件或 wxParse）
- 分享按钮 → 小程序分享卡片（带标题和封面）
- 点赞（需授权登录）
- 底部：上一篇 + 下一篇

#### 搜索
- 搜索框 + 搜索历史
- 搜索结果的文章列表
- 搜索建议（热标签展示）

#### 个人中心
- 微信一键登录 / 手机号登录
- 收藏的文章列表
- 反馈/关于

### 4.3 交互流程

```
用户打开小程序
    │
    ├── 首页加载文章列表 (GET /api/v1/articles)
    │
    ├── 点击文章 → 文章详情 (GET /api/v1/articles/{slug})
    │   ├── 点赞 → 需登录，调用 POST /api/v1/articles/{id}/like
    │   └── 分享 → 调用微信分享 API
    │
    ├── 点击分类 → 分类文章列表 (GET /api/v1/articles?category_slug=xxx)
    │
    ├── 点击标签 → 标签文章列表 (GET /api/v1/articles?tag_slug=xxx)
    │
    └── 搜索 → (GET /api/v1/articles?keyword=xxx)
```

## 5. 交互流程（关键路径）

### 5.1 用户阅读文章完整路径

```
用户点击文章链接
  │
  ├── [前端] 请求 GET /api/v1/articles/{slug}
  │   ├── 200 → 渲染文章详情页
  │   └── 404 → 显示 404 页面
  │
  ├── [前端] 用户点击点赞
  │   ├── 未登录 → 跳转登录页
  │   └── 已登录 → POST /api/v1/articles/{id}/like
  │       └── 更新 UI 点赞状态
  │
  └── [前端] 用户点击收藏
      ├── 未登录 → 跳转登录页
      └── 已登录 → POST /api/v1/articles/{id}/favorite
          └── 更新 UI 收藏状态
```

### 5.2 管理员写文章完整路径

```
管理员登录 → 进入 /admin
  │
  ├── 点击"新建文章"
  │   ├── 填写标题、内容等
  │   ├── 上传封面图 (POST /api/v1/upload/image)
  │   │   └── 返回 URL，填入封面字段
  │   ├── 编辑器内插入图片
  │   │   └── 调用上传 API → 返回 URL → 自动插入编辑器
  │   ├── [保存草稿] → POST /api/v1/articles (status: draft)
  │   │   └── 跳转到文章管理页
  │   └── [发布] → POST /api/v1/articles (status: published)
  │       └── 设置 published_at → 跳转文章管理页
  │
  └── 点击"编辑文章" (已有文章)
      ├── 请求文章数据 (GET /api/v1/articles/{slug}) 含原始 content
      ├── 预填编辑器
      ├── 修改后 [保存] → PUT /api/v1/articles/{id}
      └── 跳转回文章管理页
```

### 5.3 搜索完整路径

```
用户在搜索框输入关键词
  │
  ├── 300ms 防抖后发起请求
  │   └── GET /api/v1/articles?keyword=xxx
  │       ├── 有结果 → 展示结果列表
  │       └── 无结果 → 展示"未找到相关内容"
  │
  └── 用户点击某条结果
      └── 跳转到文章详情页
```

## 7. 后台布局（Admin Layout）

后台页面的容器布局：

```
┌──────┬───────────────────────────┐
│      │  TopBar                   │
│ Sid  │  面包屑 | 用户头像        │
│ ebar │                           │
│      ├───────────────────────────┤
│      │                           │
│  固  │  Content Area             │
│  定  │  (根据路由变化)           │
│  宽  │                           │
│  度  │                           │
│      │                           │
│      │                           │
│      │                           │
└──────┴───────────────────────────┘
```

- Sidebar: 固定宽度 240px，深色背景
- TopBar: 高度 56px，面包屑导航 + 右侧用户信息
- Content: 内边距 24px，自适应高度
- 响应式：宽屏侧边栏展开，窄屏折叠为图标或汉堡菜单
