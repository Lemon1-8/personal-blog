# 项目状态

> 当前项目：**个人博客系统**

## 需求摘要

支持 Web 端前台展示 + 后台管理 + 用户系统，部署在 Docker 中。
（移动端 App 和小程序已规划但暂不开发）

### 功能范围

| 模块 | 说明 | 端 |
|---|---|---|
| **前台展示** | 文章列表、文章详情、分类、标签、搜索 | Web |
| **后台管理** | 登录后写文章、管理分类/标签、上传图片 | Web |
| **用户系统** | 注册、登录、JWT 鉴权、角色（管理员/普通用户） | 后端统一 |

### 服务运行状态

| 服务 | 地址 | 状态 |
|---|---|---|
| 后端 API | `http://localhost:8000` (FastAPI) | ✅ 运行中 |
| 前端页面 | `http://localhost:3000` (Next.js) | ✅ 运行中 |
| API 文档 | `http://localhost:8000/docs` (Swagger) | ✅ 运行中 |
| 数据库 | PostgreSQL 16 (Docker) | ✅ 运行中 |
